# Golden output -- gerber-writer spike

Generated by `../generate.py` against the board described in `../board.yaml`.

**These goldens are only comparable at the exact library versions below.**
gerber-writer's own coordinate-format selection depends on board extents
(see REPORT.md), and any library upgrade should be treated as a reason to
regenerate + re-diff, not assume byte-compatibility.

## Pinned versions (see ../requirements.txt)

| package       | version | role                                              |
|---------------|---------|---------------------------------------------------|
| gerber-writer | 0.4.3.3 | RS-274X/X2 writer under test                       |
| pygerber      | 2.4.3   | independent X3-aware reader, used for round-trip   |
| pcb-tools     | 0.1.6   | independent (unmaintained) reader, best-effort     |

Python: 3.12.4 (Windows)

## Files

| File                | Content                                              |
|---------------------|-------------------------------------------------------|
| `board-F_Cu.gbr`    | Top copper: R1/C1 SMD pads, U1 TH pad, via pad, 2 traces |
| `board-B_Cu.gbr`    | Bottom copper: U1 TH pad, via pad, 1 trace             |
| `board-F_Mask.gbr`  | Top solder mask openings                               |
| `board-B_Mask.gbr`  | Bottom solder mask opening (U1 only)                   |
| `board-F_SilkS.gbr` | Top silkscreen: component courtyards + pin-1 tick      |
| `board-Edge_Cuts.gbr` | Board outline, 40mm x 30mm rectangle                 |
| `board-PTH.drl`     | Plated drills: U1 TH pad (0.8mm), via (0.4mm) -- hand-written Excellon |
| `board-NPTH.drl`    | Non-plated drill: 1 mounting hole (3.2mm) -- hand-written Excellon |

## Regenerating

```
cd pcb/spikes/gerber
python -m venv .venv
.venv/Scripts/pip install -r requirements.txt
.venv/Scripts/python generate.py
.venv/Scripts/python validate.py
```

## What's NOT verified here

Zero-warning visual inspection in a real Gerber viewer (gerbv or KiCad's
GerbView) is deferred to a human-in-the-loop session -- see REPORT.md,
"Remaining HITL check." `validate.py`'s structural checks and the pygerber/
pcb-tools round-trips are this round's acceptance gate, not a substitute for
that visual check.
